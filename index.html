Featured FPGA / RTL work

Native MXFP4 Gated DeltaNet decode research draft.

An FPGA-oriented research prototype for one Gated DeltaNet decode layer, focused on persistent on-chip state and native MXFP4 arithmetic.

The public evidence includes a draft PDF and datapath diagram. The draft describes FP32 and MXFP4 reference paths, deterministic test vectors, a Vitis HLS/C++ kernel structure, RTL cosimulation, and Vivado-reported implementation evidence. It does not claim production hardware, publication, or external model-quality validation.

What it is

A simulation-framed FPGA research prototype studying native MXFP4 arithmetic inside a stateful Gated DeltaNet decode path.

Tyler contributed

Reference paths, deterministic vectors, HLS structure, datapath documentation, and a draft report with explicit evidence boundaries.

Current status

Research draft/prototype. Realistic activation capture and model-quality validation remain future work.
